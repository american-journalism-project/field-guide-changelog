# AI Tools for Local News: A Landscape Guide

A living resource from [The American Journalism Project](https://theajp.org)'s Product & AI Studio.

## About

This guide helps local news outlets navigate AI tools for public meeting coverage. Each profile details what a tool does, how AI is used, who it's for, and what makes it different. Rather than making specific recommendations, we're sharing what we've learned so that newsrooms can make informed decisions.

For more on our approach and methodology, see [About This Project](About-This-Project.md).

## Structure

Each tool profile follows a consistent format:
- **Highlight** — One-line summary
- **Summary** — Brief overview of the tool and its intended users
- **What is this for?** — Core functionality and use cases
- **How is AI used?** — Details on the AI layer
- **What makes it different from other tools?** — Positioning relative to the landscape
- **Pros and Cons** — How newsrooms may assess the strengths and limitations
