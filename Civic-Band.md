# Civic Band

## Details
- URL: civic.band
- Highlight: A large datasette collection of PDF extractions from civic entities.
- Summary: CivicBand collects agendas, minutes, and other official documents that are published by civic entities and makes the full text of their contents available in a searchable and SQL accessible database. 

## What is this for?
Each CivicBand collection sits on a Datasette instance that allows basic keyword search and more complex SQL queries. Datasette is an open-source tool for exploring and publishing data. The data it is searching is raw, so ideal CivicBand users are likely technically skilled and want to use CivicBand’s output to speed up their new discovery or gathering process. 

### How is AI used?
The PDF to text conversion that is at the core of CivicBand’s system relies on Tesseract OCR (optional character recognition), a system focused on line recognition and recognizing character patterns in more than 100 languages. It also supports image formats like JPEG, PNG and TIFF. 

### What makes it different from other tools?
CivicBand focuses on published official civic documents, so it uses meeting minutes and not AI generated transcriptions from meetings (if the civic entity publishes full transcripts, those are included in CivicBand).

### Pros
- Simple Datasette interface allows for a consistent and familiar search experience for users
- CivicBand is a solo founder project who welcomes feedback and ideas from the community

### Cons
- Each civic entity is hosted separately so searching across regions does not appear possible
